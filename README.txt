Descripción(Contexto):
    Este proyecto tiene como objetivo ayudar a empresas pequeñas y medianas a tener un
    mayor control sobre sus margenes, esta idea proviene del hecho de que al yo iniciar mi empresa
    fue un problema poner en orden mis márgenes para toma de decisiones, el funcionamiento en general de
    este proyecto consiste en insertar tu base de datos en excel y con pandas leer y analizar el archivo 
    generar reportes en streamlit que muestren tus productos con mayor margen, ordena los datos y ejecuta funciones de pandas 
    para extraer margenes y mostrar tus mejores productos solo con subir la plantilla con la informacion en el formato especificado, 
    resaltara resultados fuera de lo común. Esto permitira a los usuarios tener mayor control sobre sus ventas y productos y 
    levantar banderas rojas para tomar las decisiones pertinentes.
